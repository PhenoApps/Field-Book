# <img class="icon-title" src="../_static/icons/formats/view-agenda.png"> Categorical Trait

The categorical trait format is used when trait values fall into a set of discrete categories.

### Parameters
- `Name` assign a value for trait name.
- `Details` text is displayed under the trait name on the Collect screen.
- `Categories` allows users to create unqiue categories for data collection.
- `Multiple Category Selection` allows single or multiple category selection for each entry. 
- `Unit` text denotes the unit for the observation.
- `Automatically Switch to Next Plot` toggles immediately moving to next entry when the user records an observation for an entry in the Collect screen.
- `Repeated Measures` toggles repeated measure for the trait.
- `Resource File` sets an image for the trait that will be opened by default when accessing resources from Collect.

Defined categories are displayed as buttons on the Collect screen.
